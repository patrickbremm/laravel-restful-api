## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## Patrick anotações

### Setar configuração do Sqlite inicial
DB_DATABASE=../database/database.sqlite

### Criar migration, model e controller para POST
`php artisan make:model Post -m -c --api`

### Após gerar nova model e migration, atualizar banco de dados
`php artisan migrate`

### Gerar arquivo api.php (instalação do Sanctum)
`php artisan install:api`
**Importante, quando cria o arquivo na pasta bootstrap deve ser adicionado o arquivo api.php igual o web.php

### Criar controller AuthController
`php artisan make:controller AuthController`

### Rodar
`php artisan serve`

### Criar policy para poder excluir Post
`php artisan make:policy PostPolicy`

### Criar um evento para exclusão do Post
`php artisan make:event PostDeleted`

### Criar liistener para o event PostDeleted
`php artisan make:listener LogPostDeletion`

### Criar migration para adicionar SoftDelete
`php artisan make:migration add_deleted_at_column_to_posts_table --table=posts`
`php artisan migrate`

### Criar nova migration para vincular o User ao Post
`php artisan make:migration add_user_id_to_posts_table --table=posts`

### Fazer com que o retorno do Post traga o objeto User também
`php artisan make:resource PostResource`
